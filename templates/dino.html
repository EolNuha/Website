Dinosaur Game

What is Dinosaur Game?

Dinosaur Game is like the google chrome's game when there is no internet connection. So yeah this is a recreation of that game.
To play this game use the up or space bar button to jump and the down button to crouch. There is also a pause option when the letter 'p' is hit.
You'll come to see that i've made some little changes, like tha pause option or the sound on and off option.
In this version there's also an entering window to start the game.

Link to download the application to play the game!

In the downloaded file you'll have two folders: the app and the photos folder. In the photos folder are all the game assets needed to play the game.
For the game to work the app and the photos folder need to be in the same directory.
The downloaded file will come in the form of a zip file but just extract it to a normal file.
