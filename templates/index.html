PyGames with Eol

How i started building my first games?

I have always been the kind of guy that was more into building stuff or games than into using them. And as an enthusiast i started to investigate and research different ways that i can achieve my goal and build my first game. That's how i encountered with the programing language named python. After weeks of learning i finally made it and built my first game. I called it Space Invaders. I built this game first because as a beginner someone recommended to build this as it was an easy game to built and a lot of fun to play.

In order to create this games i have had to use third party modules. I personally used pygame and it is recommended because it is the most powerful python module to build games. This module it's easy to use and has loads of powerful features. It has fast responsiveness which helps as the game is faster because of that too.

My Games and what it took to create them!

During my 3 month course with pygame i have managed to create seven games. With each of them you can basically see my progress that i've made with python and pygame. From first to the last, each of them took a lot effort and hard work, and i've learned a lot from each and every one that i've made.

Some of these games are recreations from the games that i used to play as a little boy and that i grew up playing them, others are original ideas except one named Super Mario Bros, this one i've made with a youtuber during a stream he hosted.
Every game is an offline game, meaning that internet is not needed to play the game.
