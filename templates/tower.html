Tower War

What is Tower War?

Tower War is an original game that i've based on a game that i always used to play when i was young.
This is a game that you have to buy towers and put them in strategic places that you can to most damage to monsters. There are three difficulties: 'Easy', 'Normal', 'Hard'. With each difficulty there are twenty levels, and with each level the game gets harder to beat.
Every five levels of the game, there will be a boss monster that is bigger and stronger than the rest, but so are the rewards if you manage to defeat it. In order to win you have to defeat every level in the game.

Link to download the application to play the game!

In the downloaded file you'll have two folders: the app and the photos folder. In the photos folder are all the game assets needed to play the game.
For the game to work the app and the photos folder need to be in the same directory.
The downloaded file will come in the form of a zip file but just extract it to a normal file.
